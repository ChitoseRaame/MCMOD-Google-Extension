# MCMOD-Google-Extension
MCMOD百科のChorme拓展，提供了一些微不足道的小功能。

/*------------------------------------------------------*/

目前提供的功能有：\n
1.前往百科主站\n
2.前往百科论坛\n
3.打开MC老黄历\n
4.打开MC成就生生成器\n
5.Github反馈\n
6.MCMOD快速搜索\n
7.勉励百科\n
8.关于百科\n

/*------------------------------------------------------*/

挖坑的功能：\n
1.快速登录\n
2.短消息提示 or MOD更新提示\n
3.百科主页CSS修改\n
