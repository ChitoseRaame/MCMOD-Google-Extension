# MCMOD-Google-Extension
MCMOD百科のChorme拓展，提供了一些微不足道的小功能。

/*------------------------------------------------------*/

目前提供的功能有：
1.前往百科主站
2.前往百科论坛
3.打开MC老黄历
4.打开MC成就生生成器
5.Github反馈
6.MCMOD快速搜索
7.勉励百科
8.关于百科

/*------------------------------------------------------*/

挖坑的功能：
1.快速登录
2.短消息提示 or MOD更新提示
3.百科主页CSS修改
